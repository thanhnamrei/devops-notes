# Docker - Tổng quan

## Docker là gì?
Docker là một nền tảng containerization giúp đóng gói ứng dụng cùng với tất cả dependencies của nó vào một container có thể chạy trên bất kỳ môi trường nào.

## Khi nào sử dụng Docker?

### Ưu điểm
- Consistency: Ứng dụng chạy giống nhau trên mọi môi trường
- Isolation: Cô lập ứng dụng và dependencies
- Scalability: Dễ scale và deploy
- Portability: Chạy được mọi nơi có Docker

### Sử dụng cho
- Microservices architecture
- CI/CD pipelines
- Development environment setup
- Application deployment
- Testing isolated environments

---

## Docker - Overview (English)

### What is Docker?
Docker is a containerization platform that packages applications with all dependencies into portable containers that run consistently across environments.

### Benefits
- Consistency: Same behavior across dev/staging/production
- Isolation: Dependencies and runtime are isolated per app
- Scalability: Easy to scale and deploy
- Portability: Runs anywhere Docker is available

### Use Cases
- Microservices architecture
- CI/CD pipelines
- Development environment setup
- Application deployment
- Testing in isolated environments
